std::map<std::string, bool> rosaic_node::enabled

Whether or not to publish the given mosaic message.

The key is the message name, i.e. the message ID for SBF blocks embedded in inverted commas (a string) or the message ID for NMEA messages. The value indicates whether or not to enable that message.
