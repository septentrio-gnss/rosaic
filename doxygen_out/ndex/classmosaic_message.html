const uint8_t * mosaicMessage::search ( )

Searches the buffer for the beginning of the next message (NMEA or SBF)

Returns
A pointer to the start of the next message.

Note: Will have to think about what to do in more complex case (e.g. for GPSFix, we need to store values SBF blocks to wait for others)..
