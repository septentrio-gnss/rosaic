Interprets the contents of "string" as a floating point number, stores its value in "value" and returns whether all went well.

Parameters
[in]stringThe string whose content should be interpreted as a floating point number
[out]valueThe double variable that should be overwritten by the floating point number found in "string"
Returns
True if all went fine, false if not

It checks whether or not an error occurred (via errno) and whether junk characters existed within "string", and returns true if the latter two tests are negative, false otherwise.
